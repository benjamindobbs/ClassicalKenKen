Import a Class from PowerSchool

Follow these steps to export a scores template CSV from PowerSchool:

  1. Navigate to your class in PowerSchool
  2. Create a new assignment with the settings you would like
  3. Open the assignment
  4. Click the Gear icon in the top right
  5. Select “Export Scores Template”
  6. Upload the output CSV below
