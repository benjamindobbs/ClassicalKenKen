Get Set Up in 4 Steps

Rosters are imported directly from PowerSchool using the DobbsCore Gradebook Sync Chrome extension.

  1. Install the Chrome extension
    Download the DobbsCore Gradebook Sync extension from github.com/benjamindobbs/DobbsCore and load it in Chrome via Developer Mode. See the repo README for install instructions.
  2. Copy your teacher token
    Use the button below to copy your token, then paste it into the extension popup.
  3. Configure the extension
    Click the extension icon in Chrome. Set the Server URL to https://classicaltech.org and paste your teacher token.
  4. Import your class rosters
    In PowerSchool, navigate to each of your classes. A green Import Roster to DobbsCore button will appear. Click it, enter a class name, and your roster will be imported automatically.
